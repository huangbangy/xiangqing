# 上线清单

## 1. 产品准备

- 确定小程序名称
- 确定 Logo 和简介
- 准备用户协议
- 准备隐私政策
- 准备内容审核规则
- 准备客服联系方式

## 2. 技术准备

- 微信小程序 AppID
- 后端服务
- 数据库
- 图片存储
- 管理后台
- HTTPS 域名
- 操作日志
- 备份策略

## 3. 运营准备

- 招募首批真实用户
- 准备管理员账号
- 准备审核话术
- 准备举报处理流程
- 准备封禁规则

## 4. 测试清单

- 微信登录正常
- 资料提交正常
- 审核通过和驳回正常
- 列表展示正常
- 筛选正常
- 收藏正常
- 联系申请正常
- 举报正常
- 管理后台正常
- 手机端适配正常

## 5. 上线前检查

- 不展示未审核资料
- 不公开敏感联系方式
- 不出现空白页
- 不出现错位和乱码
- 不出现无法返回的页面
- 管理后台可以追踪每一次审核动作

## 6. 最小上线标准

只要满足下面这几项，就可以先上线：

1. 用户能登录并提交资料
2. 管理员能审核资料
3. 用户能浏览公开资料
4. 用户能发起联系申请
5. 用户能举报违规内容
6. 管理员能封禁违规账号
