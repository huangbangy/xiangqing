# 页面与流程

## 1. 页面清单

### 1.1 首页/列表页

展示公开用户卡片列表。

内容：
- 红色喜庆风格头图和大“喜”字
- 今日推荐卡片
- 头像
- 昵称
- 性别
- 年龄
- 地区
- 职业
- 简短自我介绍

操作：
- 查看详情
- 收藏
- 发起联系
- 今日推荐换一批
- 标记不感兴趣
- 恢复推荐池
- 打开筛选

### 1.2 筛选页

按条件筛人。

筛选项建议：
- 性别
- 年龄区间
- 地区
- 婚姻状态
- 学历
- 职业
- 是否有小孩

### 1.3 个人资料详情页

展示完整资料。

内容：
- 头像和相册
- 基本信息
- 自我介绍
- 三句话认识我
- 择偶要求
- 联系方式保护状态
- 联系申请按钮
- 举报按钮
- 发布身份标识

### 1.4 资料编辑页

用户填写和修改资料。

状态：
- 草稿
- 待审核
- 已通过
- 已驳回
- 已下架

### 1.5 我的页

用户个人中心。

内容：
- 我的资料状态
- 资料完整度和待补充项
- 本人模式 / 家长代看模式切换
- 最近浏览
- 我的收藏
- 我发出的联系申请
- 我收到的联系申请
- 我的消息会话
- 设置

### 1.6 消息列表页

展示已通过联系申请后生成的会话。

内容：
- 对方头像、昵称和基本摘要
- 最近一条消息
- 会话时间
- 未读提示
- 本人联系 / 家长沟通标识

操作：
- 进入聊天
- 查看对方资料

### 1.7 聊天详情页

联系申请通过后的文字沟通页。

内容：
- 对方资料简卡
- 安全提醒
- 双方文字消息
- 快捷开场话术
- 输入框和发送按钮

### 1.8 举报页

填写举报原因，上传证据。

举报分类：
- 资料造假
- 骗子/广告
- 骚扰
- 色情内容
- 其他违规

### 1.9 管理后台

给运营人员和管理员使用。

功能：
- 审核资料
- 审核照片
- 处理举报
- 封禁账号
- 管理推荐位
- 查看操作日志

## 2. 用户流程

### 2.1 新用户流程

微信登录 -> 填资料 -> 查看资料完整度 -> 上传照片 -> 提交审核 -> 审核通过 -> 进入列表

### 2.2 浏览流程

首页今日推荐/列表 -> 筛选 -> 详情页 -> 收藏或联系申请

### 2.3 联系流程

用户A发起联系申请 -> 用户B收到通知 -> 用户B同意或拒绝 -> 同意后双方可查看联系方式并进入站内聊天

### 2.3.1 家长代看流程

家长切换家长代看模式 -> 浏览资料 -> 发起家长沟通申请 -> 对方同意后查看家长联系方式并进入家长沟通会话

### 2.4 举报流程

详情页点击举报 -> 填写原因 -> 提交 -> 管理后台处理 -> 结果记录

## 3. 状态定义

### 3.1 资料状态

- draft：草稿
- pending：待审核
- approved：已通过
- rejected：已驳回
- hidden：已隐藏
- banned：已封禁

### 3.2 联系申请状态

- pending：待处理
- accepted：已同意
- rejected：已拒绝
- closed：已关闭

### 3.3 会话状态

- 会话只在联系申请 accepted 后生成
- 当前 MVP 只支持文字消息
- 本地 mock 阶段不做实时推送，进入页面时刷新消息
