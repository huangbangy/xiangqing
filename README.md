# 新化相亲小程序

这是一个面向新化地区成年用户的微信相亲小程序，当前先按“能上线的最简版”规划。

## 当前已完成

- 微信小程序原生项目骨架
- 首页资料列表
- 首页“今日推荐”
- 本地筛选
- 资料详情
- 我的资料编辑与提交审核
- 资料完整度提示
- 最近浏览
- 今日推荐换一批 / 不感兴趣
- 本人模式 / 家长代看模式
- 收藏
- 联系申请
- 申请通过后的站内消息/聊天
- 举报
- 小程序内置管理后台
- 本地 mock 数据服务
- 朱砂红喜庆主题 UI

## 如何预览

1. 打开微信开发者工具
2. 选择“导入项目”
3. 项目目录选择本文件所在目录
4. AppID 可以先使用测试号，也可以把 `project.config.json` 里的 `touristappid` 换成你的真实 AppID
5. 编译后进入首页

本地演示默认使用 mock 数据，管理员口令是 `123456`。正式上线前，需要把本地 mock 数据替换成微信云开发或自有后端接口。

## MVP原则

- 只做最核心的撮合流程
- 先保证审核、举报、封禁、隐私
- 不做复杂 IM、直播、动态、付费会员
- 先用人工审核把质量控住
