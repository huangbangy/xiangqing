# 新化相亲小程序

这是一个面向新化地区成年用户的微信相亲小程序，目标是先完成“能上线的最简版”：资料建档、人工审核、浏览推荐、联系申请、站内聊天、举报处理。

## 如何预览

1. 打开微信开发者工具
2. 选择“导入项目”
3. 项目目录选择本文件所在目录
4. 确认 AppID 为 `wxfd2a7115a00c9a97`
5. 确认已开通云开发环境 `cloud1-d1gpuptkh7143c8ab`
6. 右键 `cloudfunctions/cloudPing`，选择“上传并部署：云端安装依赖”
7. 右键 `cloudfunctions/xiangqinApi`，选择“上传并部署：云端安装依赖”
8. 重新编译小程序
9. 进入“我的”页，先做云开发自检，再测试编辑资料、上传图片、提交审核、聊天

本项目保留本地 mock 兜底：云端调用失败时，部分页面仍可用本地演示数据继续预览。

## MVP 原则

- 只做最核心的撮合流程
- 先保证审核、举报、封禁、隐私
- 不做复杂 IM、直播、动态、付费会员
- 先用人工审核把资料质量控住
